## Playlist Files

Each category has its own `.m3u` playlist file located in the `playlists/` directory.

## Validation

Playlists are automatically validated daily via GitHub Actions to check for broken streams. Any issues are reported automatically.

## Usage

Download any `.m3u` file from the `playlists/` directory and import it into your IPTV player (VLC, Kodi, etc.).

## Source

Playlists are sourced from the IPTV-org project: https://github.com/iptv-org/iptv
